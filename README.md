# ziwei_skill

紫微斗数算命skill

## 项目简介

本项目是一个基于紫微斗数的命理分析技能，提供命盘分析、星曜解读、运势评估等服务。通过用户提供的出生年月日时辰和性别，生成紫微斗数命盘并进行详细分析。

## 功能概述

- 根据用户提供的出生年月日时辰和性别获取紫微斗数命盘
- 分析命盘十二宫的运势情况
- 解读命盘中的星曜组合
- 提供事业、财运、感情、健康等方面的宜忌建议
- 基于详细的星曜文档进行专业分析

## 安装和使用

### 环境要求

- Python 3.6 或更高版本
- 网络连接（用于获取命盘数据）

### 安装步骤

1. **克隆或下载项目**
   - 将项目代码克隆到本地或下载为 ZIP 文件并解压

2. **安装依赖**
   - 本项目使用 Python 标准库，无需额外安装依赖
   - 确保系统已安装 Python 3.6 或更高版本

3. **验证环境**
   ```bash
   python --version
   ```

### 使用方法

#### 1. 直接运行脚本获取命盘

```bash
# 格式：python 紫微斗数skill/scripts/get_ziwei_chart.py 出生时间 性别
# 示例：2000年10月11日14时出生的男性
python 紫微斗数skill/scripts/get_ziwei_chart.py 2000101114 M
```

**参数说明**：
- 出生时间：格式为 YYYYMMDDHH（例如：2000101114 表示2000年10月11日14时）
- 性别：M（男）或 F（女）

**返回格式**：
```json
{
  "success": true,
  "chart": [
    {
      "宫位": "命宫",
      "主星": "紫微",
      "凶星": "",
      "辅星": "",
      "状态": ""
    },
    // 其他宫位...
  ]
}
```

#### 2. 作为 Skill 使用

当用户询问紫微斗数相关问题，或提供出生年月日时辰要求分析命格时，skill 会自动触发。

**触发条件**：
- 用户询问紫微斗数相关概念、术语、原理
- 用户提供出生年月日时辰要求分析命盘
- 用户询问命宫、十二宫、星曜等具体含义

**输入示例**：
- "请帮我分析我的紫微斗数命盘，我是1990年1月1日8时出生的男性"
- "什么是紫微斗数？"
- "紫微斗数中的命宫是什么意思？"

### 打包成 .skill 文件

项目提供了打包脚本，可以将 skill 打包成 `.skill` 文件：

```bash
python package_skill.py
```

打包后会在项目根目录生成 `紫微斗数.skill` 文件，可直接用于部署。

## 分析流程

1. **命盘获取**：调用脚本获取命盘数据
2. **星曜文档读取**：读取命盘中所有星曜的详细文档
3. **命宫分析**：分析命宫及其主星和凶星
4. **身宫分析**：作为辅助推断，当命宫无主星时尤为重要
5. **三方四正分析**：分析命宫的三方四正（财帛宫、官禄宫、迁移宫）的组合情况
6. **十二宫分析**：依次分析兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、奴仆宫、官禄宫、田宅宫、福德宫、父母宫的情况
7. **星曜组合分析**：分析星曜之间的组合关系
8. **男女命差异分析**：根据男女命的不同特点进行针对性分析
9. **宜忌事项分析**：根据命盘整体情况，分析适合做什么（宜）和需要避免什么（忌）

## 注意事项

1. **时间格式**：出生时间必须严格按照 YYYYMMDDHH 格式输入
2. **性别参数**：性别必须是 M（男）或 F（女）
3. **数据来源**：命盘数据来源于第三方网站，可能存在一定的误差
4. **分析结果**：分析结果仅供参考，不构成任何决策建议
