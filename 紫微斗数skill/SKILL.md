# 紫微斗数 Skill

## 功能概述

本skill提供紫微斗数命盘分析服务，包括：
1. 根据用户提供的出生年月日时辰和性别获取命盘


## 使用方法

当用户询问紫微斗数相关问题，或提供出生年月日时辰要求分析命格时，使用此 skill。

**触发条件**：
- 用户询问紫微斗数相关概念、术语、原理
- 用户提供出生年月日时辰要求分析命盘
- 用户询问命宫、十二宫、星曜等具体含义

**输入参数**：
- 出生时间：格式为 YYYYMMDDHH（例如：2000101114 表示2000年10月11日14时）
- 性别：M（男）或 F（女）


## 脚本使用说明

### `get_ziwei_chart.py` 脚本

**功能**：获取紫微斗数命盘数据

**参数**：
- 出生时间：格式为 YYYYMMDDHH（例如：2000101114 表示2000年10月11日14时）
- 性别：M（男）或 F（女）

**使用示例**：
```bash
python 紫微斗数skill/scripts/get_ziwei_chart.py 2000101114 M
```

**返回格式**：
```json
{
  "success": true,
  "chart": { ... } // 命盘数据
}
```

## 分析流程

### ⚠️ 强制执行规则（最高优先级，必须严格遵守）
在开始任何分析之前，必须先执行以下步骤，否则不得进行任何分析：

1. **识别命盘中的所有星曜**：包括主星、吉星、凶星、辅星、煞星、四化星等所有星曜
2. **逐一读取星曜文档**：对于命盘中出现的每一个星曜，必须先使用 Read 工具读取 [星耀](references/%E6%98%9F%E8%80%80) 目录下文件名包含该星耀名的文档
3. **基于文档内容进行分析**：分析必须基于已读取的文档内容，严禁在没有读取文档的情况下直接根据星曜名称进行判断
4. **文档读取确认**：在回答用户之前，确认所有相关星曜的文档都已读取完毕，并在回答中引用文档内容

### 分析步骤
1. **命盘获取**：调用上述脚本获取命盘数据
2. **星曜文档读取**：对于命盘中出现的每一个星曜，读取 [星耀](references/%E6%98%9F%E8%80%80) 目录下文件名包含该星耀名的详细文档,分析宫位的时候先读取该宫位出现的全部星耀文档，然后基于文档内容进行分析
注意星辰后带字：
庙（灿烂）：星曜最明，得数最强，吉曜极吉，凶曜不凶。

旺（光辉）：星曜次明，得数次强，吉曜吉，凶曜不凶。

得地（光明）：星曜光明，得数适度，吉曜仍吉，凶曜不凶。

利益（尚明）：星曜尚明，得数渐弱，吉曜尚吉，凶曜渐凶。

平和（微明）：星光已低，得数已弱，吉曜力微，凶曜肆凶。

不得地（已暗）：星光已暗，得数最弱，吉曜无力，凶曜愈凶。

落陷（暗黑）：星曜无光，无数可得，吉曜无用，凶曜最凶。
3. **命宫分析**：分析命宫及其主星和凶星，这是命盘的核心，显示人的先天命运、后天命运，包括性格、品德、容貌、才能、机遇、思想、精神、爱好、适任的职业、适居的环境、一生工作和事业发展的情况、人生的顺逆等。**必须先读取命宫中所有星曜的文档，然后基于文档内容进行分析。**
4. **身宫分析**：作为辅助推断，当命宫无主星时尤为重要，身宫之星曜可视为命宫与三方四正之会合作用。**必须先读取身宫中所有星曜的文档，然后基于文档内容进行分析。**
5. **三方四正分析**：分析命宫的三方四正（财帛宫、官禄宫、迁移宫）的组合情况，这是判断一生成就的重要依据。**必须先读取三方四正中所有星曜的文档，然后基于文档内容进行分析。**
6. **十二宫分析**：依次分析兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、奴仆宫、官禄宫、田宅宫、福德宫、父母宫的情况。**每分析一个宫位，必须先读取该宫位中所有星曜的文档，然后基于文档内容进行分析。**
7. **星曜组合分析**：分析星曜之间的组合关系，包括吉星、凶星的相互作用。**必须先读取所有相关星曜的文档，然后基于文档内容分析星曜之间的组合效应。**
8. **男女命差异分析**：根据男女命的不同特点进行针对性分析，男命重事业成就，女命重家庭福份。
9. **宜忌事项分析**：根据命盘整体情况，分析适合做什么（宜）和需要避免什么（忌），包括：
    - 事业发展宜忌：根据命宫、身宫、官禄宫、迁移宫的星曜配置，分析适合的职业、发展方向、工作方式等
    - 财运管理宜忌：根据财帛宫、身宫的星曜配置，分析适合的理财方式、投资方向、守财策略等
    - 感情婚姻宜忌：根据夫妻宫、命宫的星曜配置，分析适合的结婚时机、伴侣选择、感情经营等
    - 健康养生宜忌：根据疾厄宫的星曜配置，分析需要关注的身体部位、养生方式、注意事项等
    - 人际关系宜忌：根据奴仆宫、兄弟宫的星曜配置，分析适合的交友方式、合作模式、注意事项等
    - 学习进修宜忌：根据命宫、官禄宫的星曜配置，分析适合的学习方向、进修方式、考试策略等
    - 行为举止宜忌：根据命宫、福德宫的星曜配置，分析适合的行为方式、性格调整、情绪管理等

## 注意事项

1. **时间格式**：出生时间必须严格按照 YYYYMMDDHH 格式输入
2. **性别参数**：性别必须是 M（男）或 F（女）
3. **数据来源**：命盘数据来源于第三方网站，可能存在一定的误差
4. **分析结果**：分析结果仅供参考，不构成任何决策建议
5. **文档读取要求**：**这是最重要的注意事项！在分析任何星曜之前，必须先使用 Read 工具读取 [星耀](references/%E6%98%9F%E8%80%80) 目录下文件名包含该星曜名的文档。严禁在没有读取文档的情况下直接根据星曜名称进行判断。如果文档不存在，请告知用户该星曜的文档暂时缺失。**
