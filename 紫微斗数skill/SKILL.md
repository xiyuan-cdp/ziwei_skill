# 紫微斗数 Skill

## 功能概述

本skill提供紫微斗数命盘分析服务，包括：
1. 根据用户提供的出生年月日时辰和性别获取命盘


## 使用方法


## 脚本使用说明

### `get_ziwei_chart.py` 脚本

**功能**：获取紫微斗数命盘数据

**参数**：
- 出生时间：格式为 YYYYMMDDHH（例如：2000101114 表示2000年10月11日14时）
- 性别：M（男）或 F（女）

**使用示例**：
```bash
python scripts/get_ziwei_chart.py 2000101114 M
```

**返回格式**：
```json
{
  "success": true,
  "chart": { ... } // 命盘数据
}
```

## 分析流程

### ⚠️ 强制执行规则（最高优先级，必须严格遵守）
在开始任何分析之前，必须先执行以下步骤，否则不得进行任何分析：

1. **识别命盘中的所有星曜**：包括主星、吉星、凶星、辅星、煞星、四化星等所有星曜
2. **逐一读取星曜文档**：对于命盘中出现的每一个星曜，必须先使用 Read 工具读取 `references/星耀` 目录下对应的星曜文档
3. **基于文档内容进行分析**：分析必须基于已读取的文档内容，严禁在没有读取文档的情况下直接根据星曜名称进行判断
4. **文档读取确认**：在回答用户之前，确认所有相关星曜的文档都已读取完毕，并在回答中引用文档内容

### 分析步骤
1. **命盘获取**：调用上述脚本获取命盘数据
2. **星曜文档读取**：读取命盘中所有星曜的详细文档（路径：`references/星耀`）
3. **命宫分析**：分析命宫及其主星和凶星，这是命盘的核心，显示人的先天命运、后天命运，包括性格、品德、容貌、才能、机遇、思想、精神、爱好、适任的职业、适居的环境、一生工作和事业发展的情况、人生的顺逆等。**必须先读取命宫中所有星曜的文档，然后基于文档内容进行分析。**
4. **身宫分析**：作为辅助推断，当命宫无主星时尤为重要，身宫之星曜可视为命宫与三方四正之会合作用。**必须先读取身宫中所有星曜的文档，然后基于文档内容进行分析。**
5. **三方四正分析**：分析命宫的三方四正（财帛宫、官禄宫、迁移宫）的组合情况，这是判断一生成就的重要依据。**必须先读取三方四正中所有星曜的文档，然后基于文档内容进行分析。**
6. **十二宫分析**：依次分析兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、奴仆宫、官禄宫、田宅宫、福德宫、父母宫的情况。**每分析一个宫位，必须先读取该宫位中所有星曜的文档，然后基于文档内容进行分析。**
7. **星曜庙旺利陷分析**：分析星曜的庙、旺、得地、利益、平和、不得地、落陷七种状态，这是判断星曜力量强弱的重要依据。**必须先读取相关星曜的文档，结合文档中的庙旺利陷说明进行分析。**规则如下：
   - 吉星入庙旺则为吉，吉星落陷则为凶；
   - 凶星入庙旺虽有吉意，但难减其凶，凶星落陷则属大凶；
   - 吉星入庙旺，为吉，虽不惧凶星，但仍被凶星干扰；
   - 吉星落陷，为凶，与入庙旺之凶星同宫，难减其凶，仍然属凶；有落陷之凶星同宫，大凶。
8. **星曜组合分析**：分析星曜之间的组合关系，包括吉星、凶星的相互作用。**必须先读取所有相关星曜的文档，然后基于文档内容分析星曜之间的组合效应。**
9. **四化分析**：分析化禄、化权、化科、化忌的影响，四化是紫微斗数的重要组成部分。**必须先读取相关星曜的文档以及四化星的相关文档，然后基于文档内容分析四化的影响。**
10. **男女命差异分析**：根据男女命的不同特点进行针对性分析，男命重事业成就，女命重家庭福份。
11. **人命入格判断**：根据命宫、身宫、财帛宫、官禄宫、迁移宫的吉凶情况，判断命局的格局层次，确定富贵贫贱等级。**必须基于已读取的星曜文档内容进行判断。**
12. **宜忌事项分析**：根据命盘整体情况，分析适合做什么（宜）和需要避免什么（忌），包括：
    - 事业发展宜忌：根据命宫、身宫、官禄宫、迁移宫的星曜配置，分析适合的职业、发展方向、工作方式等
    - 财运管理宜忌：根据财帛宫、身宫的星曜配置，分析适合的理财方式、投资方向、守财策略等
    - 感情婚姻宜忌：根据夫妻宫、命宫的星曜配置，分析适合的结婚时机、伴侣选择、感情经营等
    - 健康养生宜忌：根据疾厄宫的星曜配置，分析需要关注的身体部位、养生方式、注意事项等
    - 人际关系宜忌：根据奴仆宫、兄弟宫的星曜配置，分析适合的交友方式、合作模式、注意事项等
    - 学习进修宜忌：根据命宫、官禄宫的星曜配置，分析适合的学习方向、进修方式、考试策略等
    - 行为举止宜忌：根据命宫、福德宫的星曜配置，分析适合的行为方式、性格调整、情绪管理等
    - 特别时期宜忌：根据当前大运、流年的星曜配置，分析当前时期适合做什么、需要避免什么

## 注意事项

1. **时间格式**：出生时间必须严格按照 YYYYMMDDHH 格式输入
2. **性别参数**：性别必须是 M（男）或 F（女）
3. **数据来源**：命盘数据来源于第三方网站，可能存在一定的误差
4. **分析结果**：分析结果仅供参考，不构成任何决策建议
5. **文档读取要求**：**这是最重要的注意事项！在分析任何星曜之前，必须先使用 Read 工具读取 `references/星耀` 目录下对应的星曜文档。严禁在没有读取文档的情况下直接根据星曜名称进行判断。如果文档不存在，请告知用户该星曜的文档暂时缺失。**

## 常见问题

### 1. 如何计算出生时间？
- 请使用公历出生时间，精确到小时
- 例如：1990年1月1日上午8点，输入为 1990010108

### 2. 命盘分析包含哪些内容？
- 命宫及其主星分析
- 十二宫运势分析
- 星曜组合解读
- 整体运势评估

### 3. 紫微斗数与其他命理方法有什么不同？
- 紫微斗数是中国传统命理学的重要分支
- 它通过分析星曜在十二宫的分布来推断人的命运
- 与四柱八字相比，紫微斗数更加注重星曜的组合和宫位的关系

### 4. 命盘分析中的宜忌事项是什么？
- 宜忌事项是根据命盘整体情况分析出的适合做什么（宜）和需要避免什么（忌）
- 包括事业发展、财运管理、感情婚姻、健康养生、人际关系、学习进修、行为举止等方面的具体建议
- 宜忌事项结合命宫、身宫、三方四正、十二宫、星曜庙旺利陷、四化等因素综合判断
- 宜忌事项可以帮助用户更好地发挥自身优势，避免不足之处，实现人生目标

### 5. 如何根据宜忌事项调整自己的行为？
- 事业发展：根据宜忌选择适合的职业和发展方向
- 财运管理：根据宜忌选择适合的理财方式和投资方向
- 感情婚姻：根据宜忌选择合适的结婚时机和伴侣
- 健康养生：根据宜忌关注需要保养的身体部位和养生方式
- 人际关系：根据宜忌选择合适的交友方式和合作模式
- 学习进修：根据宜忌选择适合的学习方向和进修方式
- 行为举止：根据宜忌调整自己的行为方式和情绪管理
